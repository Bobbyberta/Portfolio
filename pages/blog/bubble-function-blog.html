The Making of Bubble Function: A Journey in Educational Game Design

November 10, 2019 8 min read Game Design

In 2019, I led the design and development of "Bubble Function," an educational math game that would go on to be shortlisted for a TIGA award. This is the story of how we created a game that made function machines fun and accessible for young learners.

The Challenge

When Mangahigh approached me about creating a replacement for their Algebra Meltdown game, I knew we had a significant challenge ahead. The original game had been a cornerstone of their function machine learning content, but user feedback indicated that while the core concept was solid, the difficulty curve was too steep for many students.

Initial Research

Our first step was to conduct thorough research with both teachers and students. We discovered that:

  • Students found the abstract nature of function machines challenging
  • Visual representation helped with understanding
  • Quick feedback loops were essential for learning
  • The game needed to support different learning speeds

The Design Process

We began with paper prototypes, testing different game mechanics that could make function machines more intuitive. The breakthrough came when we combined the concept with a bubble shooter mechanic - it provided a natural way to visualize input and output values while keeping the game engaging.

Key Design Decisions

  • Visual Function Machines: Made the abstract concrete
  • Progressive Difficulty: Gentle learning curve with optional challenges
  • Immediate Feedback: Clear visual and audio cues for correct/incorrect answers
  • Multiple Solution Paths: Supporting different problem-solving approaches

Technical Implementation

We built Bubble Function using HTML5 and the Phaser game engine, ensuring it would work across different devices and platforms. The technical challenges included:

  • Creating smooth animations that helped visualize the math concepts
  • Implementing precise collision detection for the bubbles
  • Optimizing performance for weaker devices
  • Ensuring consistent behavior across different screen sizes

Impact and Results

The game was well-received by both teachers and students:

  • TIGA Award shortlist for Best Educational Game
  • Increased student engagement with function machine concepts
  • Positive feedback from teachers about improved understanding
  • Successfully replaced the original game in Mangahigh's curriculum

Lessons Learned

This project reinforced several key principles of educational game design:

  • Start with the learning objectives, then build the fun around them
  • Regular testing with the target audience is crucial
  • Visual feedback helps cement mathematical concepts
  • Balance is everything - too easy is boring, too hard is frustrating
