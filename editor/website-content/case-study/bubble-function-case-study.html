Bubble Function

Games Design Release 2019

Overview

Bubble Function is a TIGA-shortlisted educational maths game designed for 9-12 year olds, aimed at improving their understanding of function machines.

HTML5 Phaser Engine Cross-platform

My Role: Design Lead

  • Concept Creation
  • Manage development of MVP
  • First iteration of visuals
  • Testing

The Challenge

Mangahigh needed a replacement for their Algebra Meltdown game. User feedback indicated that while the function machine concept was well-received, the game's overall difficulty presented a barrier to engagement.

My Approach

Understanding the User

Analyzing existing user feedback and collaborating with sales and support teams.

Conceptualization

Combining function machine concept with bubble shooter mechanic.

Rapid Prototyping

Utilizing paper prototypes and a reusable game template.

User Testing

Conducting sessions with target users in school settings.

Key Outcomes & Learnings

  • Successfully delivered a TIGA-shortlisted game
  • Improved user engagement through simplified mechanics
  • Developed reusable game template
  • Gained valuable team leadership experience
